# Adventure Works Sales Report 2022 - Power BI Dashboard

##  Project Overview

This project showcases an interactive Power BI dashboard built using **Adventure Works Sales data** for the year **2023â€“2025**. The dashboard enables detailed sales analysis by **customer, product category, sub-category, city, month**, and **year**, helping stakeholders derive business insights such as:

- Sales trends over time
- Year-over-year (YoY) performance
- Sales performance against budget targets
- Product and customer segmentation

The goal of this dashboard is to demonstrate the power of **data storytelling** with interactive visuals, slicers, and DAX calculations to track **performance**, identify **opportunities**, and make **data-driven decisions**.
